關於我們

DevOps Taiwan 社群成立于 2015/5/21，由多位關注 DevOps 相關資訊的 IT、軟體界朋友一起共同創立，期望透過社群互動的方式建立一個平台提供台灣的社群朋友可以一起針對 DevOps 相關的議題、專案、技術與工作進行分享與交流。

社群目前主要活動於 Facebook 社團

特別合作

來自香港的合作訊息，提供給有興趣的社群朋友們參考，DevOps Taiwan 的社群朋友如欲前往此展會擺攤、參訪，可透過 DevOps Taiwan 轉介香港的負責窗口，可獲得額外的優惠。

Cloud Expo Asia, Hong Kong（香港亞太雲端科技博覽）

https://www.cloudexpoasiahk.com/event-listing-2018

2018 年 5 月，將舉辦第三屆的 Cloud Expo Asia, Hong Kong，在這場大型展會中，預計將會有：

  • 10,000名與會嘉賓──2017年吸引了8879名（數字經第三方審核），例如：百度、匯豐銀行、國泰航空、香港警務署、TVB。
  • 250位政商屆的頂尖演講嘉賓──例如：9GAG、CA Technologies、餓了麼、微博、蘭桂坊集團、香港機場管理局、瑞士信貸集團、GoGoVan、SenseTime。
    詳情：www.cloudexpoasiahk.com/conference-programme
  • 200個行業領先的國際參展商──例如：阿里雲、甲骨文、騰訊雲、AWS、Carbon Black、McAfee、Veeam。 詳情：www.cloudexpoasiahk.com/exhibitors
  • 100個國際和區域性合作夥伴──例如：Cloud Security Alliance、(ISC)2、DevOps Taiwan Community、SuperCharger、前海、深港科技合作促進會、深圳市計算機用戶協會、香港電腦學會。
  • 多個跨主題的活動亮點──例如：由Chef、Docker、IBM和Viseo帶領的互動式研討會「DevOps Live！」，大數據及人工智能（Big Data & A.I.），金融科技（FinTech），主機代管（Colocation），現場演示實時運作的數據中心（Live Data Centre）等等。
